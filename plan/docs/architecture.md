# 系统架构

## 整体架构

```
┌─────────────────────────────────────────────────────────────────┐
│                         应用层 (Application)                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐ │
│  │   Client    │  │   Server    │  │      P2P Ring Node      │ │
│  │  (成员A)    │  │  (成员B)    │  │        (成员B)          │ │
│  └─────────────┘  └─────────────┘  └─────────────────────────┘ │
├─────────────────────────────────────────────────────────────────┤
│                         模型层 (Model)                           │
│  ┌─────────────┐  ┌─────────────┐                              │
│  │   Model     │  │   Dataset   │                              │
│  │  (成员A)    │  │  (成员A)    │                              │
│  └─────────────┘  └─────────────┘                              │
├─────────────────────────────────────────────────────────────────┤
│                         协议层 (Protocol)                        │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐ │
│  │   Message   │  │   Codec     │  │     Serializer          │ │
│  │  (成员C)    │  │  (成员C)    │  │       (成员C)           │ │
│  └─────────────┘  └─────────────┘  └─────────────────────────┘ │
├─────────────────────────────────────────────────────────────────┤
│                         传输层 (Transport)                       │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐ │
│  │ Connection  │  │   Listener  │  │     Timeout Control     │ │
│  │  (成员C)    │  │  (成员C)    │  │       (成员C)           │ │
│  └─────────────┘  └─────────────┘  └─────────────────────────┘ │
├─────────────────────────────────────────────────────────────────┤
│                         工具层 (Utils)                           │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────────┐ │
│  │   Logger    │  │   Metrics   │  │     Config Loader       │ │
│  │  (成员D)    │  │  (成员D)    │  │       (成员D)           │ │
│  └─────────────┘  └─────────────┘  └─────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## 模块职责

### 成员A：模型、数据、客户端

| 模块 | 文件 | 职责 |
|------|------|------|
| Model | `model/base.py`, `model/simple_cnn.py` | 模型定义、权重存取、训练、评估 |
| Data | `data/dataset.py`, `data/partitioner.py` | 数据集加载、划分策略 |
| Client | `client/federated_client.py`, `client/trainer.py` | 客户端连接、模型接收、本地训练、更新发送 |
| Config | `config/*.yaml` | 配置文件维护 |

### 成员B：服务端开发

| 模块 | 文件 | 职责 |
|------|------|------|
| Server | `server/federated_server.py` | 服务端主逻辑、多线程管理 |
| Aggregator | `server/aggregator.py` | 聚合算法（FedAvg等） |
| ClientManager | `server/client_manager.py` | 客户端连接管理 |
| RoundCoordinator | `server/round_coordinator.py` | 轮次协调、超时控制 |
| P2P | `p2p/ring_node.py`, `p2p/topology.py` | 环形拓扑、故障检测与恢复 |

### 成员C：网络协议设计

| 模块 | 文件 | 职责 |
|------|------|------|
| Message | `protocol/message.py` | 消息结构定义 |
| Codec | `protocol/codec.py` | 编解码、粘包处理 |
| Serializer | `protocol/serializer.py` | 模型权重序列化 |
| Transport | `transport/connection.py`, `transport/listener.py` | TCP连接封装、发送/接收 |

### 成员D：测试与日志

| 模块 | 文件 | 职责 |
|------|------|------|
| Logger | `utils/logger.py` | 结构化日志输出 |
| Metrics | `utils/metrics.py` | 指标统计、准确率曲线 |
| Timer | `utils/timer.py` | 耗时统计 |
| Tests | `tests/` | 单元测试、集成测试 |
| Docs | `docs/` | 测试报告、用户手册 |

## 数据流

### 中心化模式

```
┌──────────────────────────────────────────────────────────────┐
│                        Round N                                │
├──────────────────────────────────────────────────────────────┤
│  Server                          Client 1, 2, 3              │
│    │                                  │                       │
│    │──── MODEL_BROADCAST ────────────>│  (广播全局模型)        │
│    │                                  │                       │
│    │                          [本地训练]                      │
│    │                                  │                       │
│    │<──── MODEL_UPDATE ───────────────│  (上传更新)           │
│    │<──── MODEL_UPDATE ───────────────│                       │
│    │<──── MODEL_UPDATE ───────────────│                       │
│    │                                  │                       │
│  [聚合]                                │                       │
│    │                                  │                       │
└──────────────────────────────────────────────────────────────┘
```

### 去中心化模式（环形）

```
┌──────────────────────────────────────────────────────────────┐
│                        Round N                                │
├──────────────────────────────────────────────────────────────┤
│                                                               │
│    Node A ───── RING_PASS ────> Node B                       │
│      ↑                            │                          │
│      │                            ↓                          │
│      │                        [本地训练]                      │
│      │                            │                          │
│      │                            ↓                          │
│      │                        Node C                         │
│      │                            │                          │
│      │                            ↓                          │
│      │                        [本地训练]                      │
│      │                            │                          │
│      │                            ↓                          │
│      │                        Node A                         │
│      │                            │                          │
│      │<───── RING_PASS ───────────┘                          │
│      │                                                       │
│  [融合收到的模型]                                              │
│                                                               │
└──────────────────────────────────────────────────────────────┘
```

## 状态机

### 客户端状态

```
┌─────────┐    connect()    ┌─────────┐    receive_model()    ┌─────────┐
│  IDLE   │ ───────────────>│ CONNECT │ ─────────────────────>│ TRAIN   │
└─────────┘                 └─────────┘                       └─────────┘
     ↑                                                            │
     │                                                            │
     │                      send_update()                         │
     └────────────────────────────────────────────────────────────┘
```

### 服务端状态

```
┌─────────┐    start()    ┌─────────┐    wait_clients()    ┌─────────┐
│  IDLE   │ ─────────────>│ LISTEN  │ ───────────────────>│  WAIT   │
└─────────┘               └─────────┘                      └─────────┘
                                                               │
    ┌──────────────────────────────────────────────────────────┘
    │
    ↓
┌─────────┐    broadcast()    ┌─────────┐    collect()    ┌─────────┐
│ BROADCAST│ ────────────────>│ COLLECT │ ──────────────>│ AGGREGATE│
└─────────┘                   └─────────┘                 └─────────┘
                                                               │
    ┌──────────────────────────────────────────────────────────┘
    │
    ↓
┌─────────┐
│  NEXT   │ ───> BROADCAST (下一轮)
│  ROUND  │
└─────────┘
```
